Банк Аналитик — Система семантического поиска по отчетам Национального банка
Данный проект представляет собой локальную интеллектуальную поисковую систему (прототип RAG-архитектуры без генеративного LLM-модуля), спроектированную для автоматического анализа, индексации и семантического поиска информации по сложным аналитическим документам и финансовым отчетам Национального банка.
Проект построен на базе Streamlit (для веб-интерфейса), FAISS (для сверхбыстрого векторного поиска) и Sentence-Transformers (для формирования многоязычных смысловых эмбеддингов).
Архитектура и стек технологий
Проект реализует эффективный конвейер обработки и поиска данных:
Извлечение и препроцессинг (extract_text.py, preprocess.py): Извлечение сырого текста из отчетов и очистка данных.
Грануляция (chunks/): Нарезка документов на небольшие логические фрагменты (чанги) для точечного поиска.
Эмбеддинг-модель: Передовая многоязычная модель paraphrase-multilingual-MiniLM-L12-v2 от Sentence-Transformers, которая превосходно понимает контекст и синонимы на русском и других языках.
Векторный индекс (faiss_index.bin): База данных FAISS (библиотека от Meta), вычисляющая сходство по расстоянию L2 (Евклидово расстояние) за миллисекунды.
Пользовательский интерфейс (app.py): Минималистичное веб-приложение на Streamlit для ввода вопросов и отображения релевантных выдержек из документов.

Инструкция по развертыванию и запуску
1. Клонирование и подготовка окружения
Убедитесь, что у вас установлен Python 3.9+. Рекомендуется использовать виртуальное окружение:
Bash
# Активация виртуального окружения (если venv уже создан)
source venv/bin/activate  # Для Linux/macOS
# или
venv\Scripts\activate     # Для Windows

# Установка всех необходимых зависимостей
pip install -r requirements.txt
2. Сборка векторного индекса (Индексация)
Перед запуском приложения необходимо наполнить папку chunks/ текстовыми фрагментами отчетов (если они не подготовлены). Затем запустите скрипт генерации векторной базы данных:
Bash
python create_index.py
Что произойдет: Скрипт загрузит модель paraphrase-multilingual-MiniLM-L12-v2, переведет все текстовые чанки в векторы размерности 384, создаст эффективный индекс faiss_index.bin и сопутствующий файл соответствия texts.txt.
3. Запуск веб-приложения
После успешной генерации индекса запустите графический интерфейс Streamlit:
Bash
streamlit run app.py
После запуска консоль выдаст локальный адрес (обычно http://localhost:8501), который откроется автоматически в вашем браузере.
Разбор ключевых модулей системы
create_index.py (Сборка базы знаний)
Читает файлы из папки chunks/ в алфавитном порядке, векторизует их с помощью эмбеддингов HuggingFace, инициализирует индекс faiss.IndexFlatL2` и сохраняет готовую структуру на диск. Дополнительно убирает переносы строк внутри фрагментов для красивого отображения в UI.
qa_system.py (Ядро поиска)
Реализует функцию get_answer(question, k=3).
Кодирует входящий запрос пользователя в вектор.
Осуществляет поиск k ближайших векторов-соседей в пространстве FAISS.
Дедуплицирует найденные фрагменты (чтобы избежать выдачи одинакового контента) и форматирует результат в удобный markdown-блок.
app.py (Интерфейс пользователя)
Обеспечивает удобный UI-слой: поле ввода вопроса, индикатор загрузки (st.spinner) во время работы нейросети, блок вывода ошибок в случае сбоев и контейнер st.info для демонстрации релевантных параграфов из отчетов Нацбанка.
